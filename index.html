CSS - Cascading Style Sheets

Flexbox

Container:

  • class="container"
  • display: flex OR inline-flex -> applies the flexbox rules
    • flex: flexbox width is the full width of the page
    • inline-flex: flexbox width is as big as its biggest inner element's width
  • flex-direction: determines which direction the flexbox is stacked
    • row: default, main-axis is the horizontal axis
    • column: vertically stacked, main-axis is the vertical axis
  • flex-wrap: determines behavior when items overflow page
    • nowrap: default, containues off the page
    • wrap: children move to next line
    • wrap-reverse: children move from bottom right to left
  • gap: 1rem; -> adds a gap between each of the cards
  • justify-content: sets distribution of children across the main axis
    • flex-start: default
    • flex-end: starts from the right
    • center: center all the childrens?
    • space-between: only spaces between children
    • space-around: spaces around all the children

Children:

  • flex-basis: determines how long to spread across main-axis
    • if main-axis is horizontal-axis: flex-basis increases the width of the children
    • main is vertical: flex-basis changes height of children
  • order: set order of children
